Predict ATT&CK is a tool for predicting ATT&CK tactics and techniques from cyber threat reports. Paste your report in the text area to see predictions. Darker backgrounds indicate predicted tactics/techniques with a displayed likelihood. If a tactic/technique is shown as absent despite the likelihood, it's due to model post-processing. You can correct predictions and save them to the training set to improve accuracy.
